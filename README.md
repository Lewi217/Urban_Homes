# 🏡 Urban Homes

> **An investment in homes** — A modern real estate web application built with Next.js, TypeScript, and Tailwind CSS.

---

## 📌 Overview

Urban Homes is a full-featured real estate platform designed to help users discover, explore, and invest in residential properties. Built with a clean, scalable architecture using the latest Next.js App Router, it delivers a fast and responsive experience across all devices.

---

## 🚀 Tech Stack

| Technology | Purpose |
|---|---|
| **Next.js 14+** | React framework with App Router |
| **TypeScript** | Type-safe development |
| **Tailwind CSS** | Utility-first styling |
| **PostCSS** | CSS processing pipeline |

---

## 📁 Project Structure

```
Urban_Homes/
├── app/               # Next.js App Router pages & layouts
├── components/        # Reusable UI components
├── context/           # React Context providers (global state)
├── lib/               # Library configs & shared utilities
├── services/          # API calls & data-fetching logic
├── types/             # TypeScript type definitions
├── utils/             # Helper functions
├── public/images/     # Static image assets
├── tailwind.config.ts # Tailwind configuration
├── next.config.mjs    # Next.js configuration
└── tsconfig.json      # TypeScript configuration
```

## ⚙️ Getting Started

### Prerequisites

- **Node.js** v18 or higher
- **npm** or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/Lewi217/Urban_Homes.git

# Navigate into the project
cd Urban_Homes

# Install dependencies
npm install
```

### Running the Development Server

```bash
npm run dev
```

Or using the provided script:

```bash
bash start-dev.sh
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

### Building for Production

```bash
npm run build
npm start
```
